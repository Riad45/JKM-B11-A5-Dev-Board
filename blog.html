What are the different ways to select an element in the DOM?
We can select an element in the DOM using methods like getElementById() for selecting by ID, getElementsByClassName() for selecting by class name, getElementsByTagName() for selecting by tag name, querySelector() for selecting the first matching element using CSS selectors, and querySelectorAll() for selecting all matching elements.
Difference between innerHTML, innerText, and textContent?
innerHTML gets or sets the HTML content inside an element. innerText gets or sets only the visible text of an element, considering CSS styling. textContent gets or sets all the text content, regardless of CSS styling, including hidden elements.
What is event delegation?
Event delegation is a technique where we attach a single event listener to a parent element, and the event listener handles events for all its child elements, even if they are dynamically added or removed.
What is event bubbling?
Event bubbling is the process where an event starts from the target element and bubbles up to its ancestors in the DOM hierarchy, triggering any relevant event listeners on each ancestor along the way.
How to create, add, and remove elements in JavaScript?
To create an element, we use document.createElement(). To add an element, we can use appendChild() or insertBefore(). To remove an element, we use the remove() method.
